* Visual on files:
- U = Untracked (File mới, chưa được git theo dõi)
- A = Added (Đã thêm vào staging)
- D = Deleted (Đã xóa)
- R = Renamed (Đã đổi tên)
- M = Modified (Đã sửa đổi)
File đã được thay đổi so với commit cuối cùng
Chưa được git add
